Nouveau portail de demande de résidence permanente en ligne pour les représentants autorisés

Immigration, Réfugiés et Citoyenneté Canada (IRCC) lancera une nouvelle résidence permanente en ligne Portail d'application (portail d'application en ligne PR), permettant une immigration autorisée (payante) représentants de soumettre des demandes de résidence permanente (non-entrée express) en ligne au nom de leurs clients. Après le lancement en douceur et les tests initiaux du produit viable minimal pour clients uniquement en mars 2021, le portail de candidature en ligne PR sera disponible pour les représentants d'ici septembre 2021.

Les représentants pourront utiliser le portail de candidature en ligne PR pour gérer les candidatures en ligne à partir d'un seul compte pour plusieurs clients pour les volets de demande de résidence permanente, qui ont été sur papier. Les candidatures papier resteront disponibles pour ceux qui ont besoin d'un logement. Les Le portail de candidature en ligne PR ne sera pas connecté au portail des représentants rémunérés autorisés.

Pour créer un compte PR Online Application Portal, les représentants devront fournir des informations, y compris leur numéro d'identification de membre, leur adresse professionnelle et leur nom, et ils devront télécharger un copie valide d'une pièce d'identité (passeport, permis de conduire ou carte de séjour permanent en cours de validité).

Que signifierait une victoire conservatrice aux élections fédérales canadiennes pour l'immigration?

Avec le développement d'un flux de traitement accéléré pour ceux qui en ont les moyens, les temps d'attente pour les demandes d'immigration seraient considérablement réduites sous un gouvernement conservateur.

Dans sa plate-forme électorale Secure The Future, le Parti conservateur d'Erin O'Toole promet que « tous les revenus de ces frais de traitement accéléré seraient utilisés pour embaucher des employés supplémentaires pour rationaliser encore plus le traitement des files d'attente"

Selon le parti, un traitement plus rapide pour les personnes qui paient pour un service accéléré sera également éliminer les candidatures de la file d'attente habituelle, réduisant ainsi les temps d'attente pour tous les candidats.
